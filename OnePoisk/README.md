# OnePoisk — Поисковая система нового поколения

![OnePoisk Logo](assets/default-avatar.svg)

> **Найдётся всё!** — современная поисковая система с интерфейсом в стиле Яндекс

## 🚀 Особенности

- 🔍 **Поиск** — быстрый и удобный поиск информации
- 🎤 **Голосовой ввод** — поиск голосом как в Яндекс
- 👤 **Oink ID** — локальная система профилей с аватарами, никнеймами и монетами
- 🕶️ **Режим инкогнито** — приватный browsing без сохранения истории
- 🔖 **Закладки** — сохранение найденных сайтов
- 🪙 **One Pro** — подписка без рекламы на 14 дней
- 🌐 **Сервисы** — доступ к популярным сервисам (Почта, Карты, Маркет и др.)
- 📱 **Адаптивный дизайн** — работает на всех устройствах

## ⚙️ Конфигурация

### Добавление сервисов

Откройте `js/main.js` и найдите массив `services`. Добавьте новый сервис:

```javascript
{
    id: 'newservice',      // Уникальный ID
    name: 'Название',      // Название сервиса
    icon: '🎯',            // Эмодзи или символ
    url: 'https://...'     // URL для открытия
}
```

### Географические ограничения

В `js/main.js` измените массив `allowedCountries` для настройки доступа по странам:

```javascript
const allowedCountries = [
    'RU', // Россия
    'BY', // Беларусь
    'KZ', // Казахстан
    // ... другие страны
];
```

### One Pro
- **Цена:** 100 монет
- **Срок:** 14 дней

## 🎤 Голосовой поиск

Использует Web Speech API браузера. Поддерживается в:
- Google Chrome
- Microsoft Edge
- Safari (частично)

## 🔒 Приватность

Все данные хранятся **локально** в браузере пользователя (localStorage):
- Аватар профиля
- Никнейм
- Баланс монет
- Закладки
- Настройки

Данные **не передаются** на серверы.

## 📢 Реклама

Рекламные блоки интегрированы через скрипт:
```html
<script type="text/javascript" src="https://vak345.com/s.js?v=03208824be369b060dba1f2083d6a4c" async></script>
```

При активной подписке One Pro реклама автоматически скрывается.
