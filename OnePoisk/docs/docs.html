Документация OnePoisk

🚀 Быстрый старт

OnePoisk — это современная поисковая система с интерфейсом в стиле Яндекс. Все файлы находятся в папке проекта и готовы к развёртыванию.

Структура проекта:

OnePoisk/ ├── index.html # Главная страница ├── landing.html # Лендинг ├── css/ │ └── style.css # Стили ├── js/ │ └── main.js # Логика и сервисы ├── assets/ # Изображения и логотипы └── docs/ # Документация ├── faq.html ├── policy.html ├── terms.html └── docs.html

🔧 Добавление сервисов

Сервисы добавляются через редактирование файла js/main.js. Найдите массив services и добавьте новый объект:

{ id: 'newservice', // Уникальный ID name: 'Название', // Название сервиса icon: '🎯', // Эмодзи или символ url: 'https://...' // URL для открытия }

После добавления сервиса обновите страницу — он появится в сетке сервисов.

🎤 Голосовой поиск

Голосовой поиск использует Web Speech API браузера. Поддерживается в Chrome, Edge и других современных браузерах.

  • Нажмите на кнопку микрофона
  • Разрешите доступ к микрофону
  • Произнесите запрос
  • Поиск выполнится автоматически

👤 Oink ID

Система профилей пользователей с локальным хранением данных:

  • Аватар: Загружается через кнопку 📷 и сохраняется в localStorage
  • Никнейм: Вводится в текстовое поле
  • Монеты: Начисляются за ежедневный вход (10 монет/день)
  • One Pro: Подписка за 100 монет на 14 дней без рекламы
  • Закладки: Сохранение найденных сайтов

🕶️ Режим инкогнито

Включается кнопкой с маской в шапке. В этом режиме:

  • Данные не сохраняются в Oink ID
  • Тёмная тема интерфейса
  • История не ведётся

🌍 Географические ограничения

Доступ разрешён только из стран РФ и СНГ. Проверка происходит через API ipapi.co.

Список разрешённых стран: RU, BY, KZ, KG, TJ, AM, AZ, MD, UZ, TM, GE, UA

📢 Реклама

Рекламные блоки интегрированы в следующих местах:

  • Верхний блок (topAdPad) — под шапкой
  • Контентный блок (content-ad) — после сервисов

При активной подписке One Pro реклама скрывается автоматически.

🔖 Закладки

Кнопка "🔖 В закладки" появляется в правом нижнем углу при открытом сайте в модальном окне. Закладки сохраняются в Oink ID и доступны через модальное окно закладок.

🚀 Развёртывание

Vercel:

1. Залейте проект на GitHub 2. Импортируйте репозиторий в Vercel 3. Deploy автоматически

Tatnet и другие платформы:

Просто загрузите все файлы проекта на хостинг. Проект статический и не требует серверной части.

  • CSS3 — современные стили с Flexbox и Grid
