Политика конфиденциальности

1. Общие положения

OnePoisk уважает конфиденциальность пользователей. Все данные хранятся локально в браузере пользователя и не передаются на сторонние серверы.

2. Сбор данных

Мы собираем только следующие данные, которые хранятся исключительно в вашем браузере:

  • Аватар профиля (загружается пользователем)
  • Никнейм
  • Баланс монет
  • Закладки
  • Настройки режима инкогнито

3. Использование данных

Все собранные данные используются исключительно для улучшения пользовательского опыта и функционирования поисковой системы.

4. Защита данных

Данные хранятся в локальном хранилище браузера (localStorage) и доступны только с вашего устройства. При очистке кеша браузера данные будут удалены.

5. Сторонние сервисы

OnePoisk использует сторонние рекламные сети и сервисы для определения страны пользователя. Эти сервисы могут собирать свои собственные данные согласно их политикам конфиденциальности.

6. Изменения в политике

Мы оставляем за собой право вносить изменения в данную политику конфиденциальности. Актуальная версия всегда доступна на этой странице.

7. Контакты

По вопросам конфиденциальности обращайтесь через форму обратной связи на сайте.
