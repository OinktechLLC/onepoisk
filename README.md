## 🌟 Особенности

### 🔍 Поиск и навигация
- **Умный поиск** — интеграция с Wikipedia и DuckDuckGo
- **Голосовой ввод** — как в Яндексе (Web Speech API)
- **Алиса-помощник** — улучшенный голосовой ассистент
- **Быстрые ссылки** — тренды, ТВ, музыка, кино, курсы валют

### 🎨 Дизайн и интерфейс
- **Стиль ya.ru** — один в один как у Яндекса
- **Живые обои** — анимированный градиент с интерактивностью
- **Виджет погоды** — реальная погода через Open-Meteo API
- **Лента рекомендаций** — аналог Дзен с новостями
- **Адаптивный дизайн** — работает на всех устройствах

### 👤 Oink ID (аналог Яндекс ID)
- **Локальные аккаунты** — хранение в localStorage
- **Аватарки и ники** — загрузка и сохранение
- **Монеты** — ежедневная награда +10 монет
- **Подписка One Pro** — 99 монет, 14 дней без рекламы
- **Синхронизация** — между вкладками браузера

### 🔒 Безопасность и приватность
- **Геоблокировка** — только РФ и СНГ (RU, BY, KZ, AM, AZ, KG, MD, TJ, TM, UZ)
- **Режим инкогнито** — приватный поиск без истории
- **Турбо-режим** — ускоренная загрузка страниц

### 🛠️ Функционал
- **Сервисы** — 14 сервисов как у Яндекса (Почта, Карты, Маркет, Музыка, Видео и др.)
- **Закладки** — сохраняйте найденное
- **Модальные окна** — все сайты открываются внутри приложения
- **Уведомления** — красивые toast-уведомления
- **Статистика** — подсчёт поисковых запросов

### 💰 Монетизация
- **Реклама** — интегрирована vak345.com как в Яндексе
- **Подписка One Pro** — платная подписка без рекламы
- **Ежедневные награды** — система лояльности

## 🎯 Функционал

### Поисковая система

- Поиск через Wikipedia API
- Интеграция с DuckDuckGo Instant Answer
- Результаты открываются в модальном окне
- Кнопка "Мне повезёт!" (как у Google)
- Быстрые подсказки и тренды

### Голосовой поиск и Алиса

```javascript
// Работает в Chrome, Edge, Safari
const recognition = new SpeechRecognition();
recognition.lang = 'ru-RU';
recognition.interimResults = true; // Промежуточные результаты
```

**Улучшения:**
- Анимация кнопки при прослушивании
- Отображение промежуточных результатов
- Автоматический запуск поиска после завершения речи
- Уведомления о статусе распознавания

### Живые обои и погода

- **Анимированный градиент** — плавная смена цветов
- **Интерактивность** — реакция на движение мыши
- **Виджет погоды** — реальная температура через Open-Meteo API
- **Определение города** — случайный город из списка крупных городов РФ

### Лента рекомендаций (Дзен)

- 8 новостных карточек с разными темами
- Автоматическая генерация обложек с градиентами
- Hover-эффекты и анимации
- Клики обрабатываются через уведомления

### Oink ID

Локальная система аккаунтов с хранением в localStorage:

- Регистрация/Вход
- Загрузка аватара (сохраняется до очистки кеша)
- Никнеймы
- Монеты (ежедневная награда +10)
- Подписка One Pro (99 монет, 14 дней, без рекламы)

### Закладки

- Автоматическое сохранение открытых сервисов
- Ручное управление
- Хранение в localStorage

### Сервисы

По умолчанию включены:
- Почта, Карты, Маркет, Музыка
- Видео, Новости, Переводчик, Диск
- Телеграм, ВКонтакте, Одноклассники
- Афиша, Погода, Пробки

**Добавление новых сервисов:**

Откройте `app.js` и добавьте в массив `defaultServices`:

```javascript
{ 
    name: 'Название', 
    url: 'https://example.com', 
    icon: '🎯', 
    color: '#ff0000' 
}
```

## ⚠️ Важные заметки

1. **Oink ID хранит данные локально** — при очистке кеша данные удаляются
2. **Геоблокировка работает через IP** — используется API ipapi.co
3. **Голосовой поиск** — требует HTTPS или localhost
4. **Реклама** — может не работать с блокировщиками
5. **Сервисы в iframe** — некоторые сайты могут блокировать отображение в iframe
6. **Погода** — требует подключения к Open-Meteo API

## 🎨 Добавление новых сервисов

Откройте `app.js` и найдите массив `defaultServices`:

```javascript
const defaultServices = [
    { name: 'Почта', url: 'https://mail.ru', icon: '📧', color: '#005ff9' },
    // Добавьте свой сервис:
    { name: 'Название', url: 'https://example.com', icon: '🎯', color: '#ff0000' }
];
```

Просто добавьте новый объект в массив, и сервис появится на главной странице!

## 🚀 Новые функции (Яндекс-браузер стиль)

### Турбо-режим
Ускоренная загрузка страниц через прокси-сервер:
```javascript
localStorage.setItem('onepoisk_turbo', 'true');
location.reload();
```

### Синхронизация между вкладками
Oink ID автоматически синхронизируется между всеми открытыми вкладками.

### Статистика использования
В консоли доступна статистика:
```javascript
console.log('OnePoisk Stats:', JSON.parse(localStorage.getItem('onepoisk_stats')));
```
