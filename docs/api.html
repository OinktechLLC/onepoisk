API документация

Vilak.tv API позволяет встраивать плеер и получать информацию об источниках программно.

Базовый URL

https://vilak.tv

Плеер (основной endpoint)

GET/{kpId}

Открывает плеер с поиском по ID Кинопоиска.

Параметры запроса

kpIdID фильма на Кинопоиске (в пути URL)
?typefilm | serial
?sНомер сезона (для сериалов)
?eНомер серии (для сериалов)
?autoplay1 — автозапуск

Примеры URL

https://vilak.tv/301 # Фильм ID 301 https://vilak.tv/77044?type=serial # Сериал https://vilak.tv/77044?type=serial&s=2&e=5 # Сезон 2, Серия 5 https://vilak.tv/301?autoplay=1 # С автозапуском

JavaScript API (клиентский)

После подключения vilak-embed.js доступна глобальная функция Vilak():

JavaScript
// Создать плеер const player = Vilak(selector, options); // Методы объекта player: player.setSeason(seasonNumber); // переключить сезон player.setEpisode(season, episode); // переключить серию player.destroy(); // удалить плеер

Внутренний поисковой робот

При загрузке плеера автоматически запускается поисковой робот VilakPlayer (доступен внутри iframe):

JavaScript (внутри плеера)
// Инициализировать поиск по ID VilakPlayer.init(kpId); // Получить текущее состояние const state = VilakPlayer.getState(); // { kpId, type, season, episode, sourceId, sourceResults, ... } // Запустить поиск по конкретному источнику VilakPlayer.runSearchRobot(kpId, type, season, episode, 'youtube'); // Переключить источник VilakPlayer.selectSource('rutube');
