⚡ Быстрый старт

Минимальный код для встраивания плеера с ID 301 (Матрица):

HTML
<div id="player"></div> <script src="https://vilak.tv/vilak-embed.js"></script> <script> Vilak('#player', { kpId: 301 }); </script>

Способ 1 — vilak-embed.js Рекомендуется

Наиболее гибкий способ. Подключите скрипт один раз и создавайте плееры через JS.

Фильм

JavaScript
Vilak('#player', { kpId: 301, // ID Кинопоиска width: '100%', // ширина });

Сериал (конкретная серия)

JavaScript
Vilak('#player', { kpId: 77044, // ID сериала type: 'serial', season: 2, episode: 5, });

Способ 2 — iframe

Простейший способ — работает в любом конструкторе сайтов (Tilda, Wix, WordPress и т.д.).

HTML
<iframe src="https://vilak.tv/301" width="100%" height="500" frameborder="0" allowfullscreen allow="autoplay; fullscreen; picture-in-picture" ></iframe>

Способ 3 — data-атрибуты

Разместите div с атрибутами прямо в HTML — скрипт автоматически инициализирует плееры при загрузке.

HTML
<!-- Фильм --> <div class="vilak-auto" data-kpid="301"></div> <!-- Сериал --> <div class="vilak-auto" data-kpid="77044" data-type="serial" data-season="1" data-episode="1" ></div> <!-- Скрипт один раз в конце страницы --> <script src="https://vilak.tv/vilak-embed.js" async></script>

Формат URL

Vilak.tv принимает ID Кинопоиска напрямую в пути:

URL-формат
https://vilak.tv/{kpId} https://vilak.tv/{kpId}?type=serial&s=1&e=1 https://vilak.tv/{kpId}?autoplay=1

Примеры: vilak.tv/301 · vilak.tv/77044?type=serial&s=2&e=5

Параметры

ПараметрТипОбязательныйОписание
kpId number | string ✦ Да ID фильма или сериала на Кинопоиске
type string Нет 'film' (default) или 'serial'
season number Нет Номер сезона (для сериалов). Default: 1
episode number Нет Номер серии (для сериалов). Default: 1
width string Нет Ширина плеера. Default: '100%'
height number Нет Высота в px (отключает responsive режим)
responsive boolean Нет Адаптивная высота 16:9. Default: true
autoplay boolean Нет Автозапуск. Default: false

JavaScript API

Функция Vilak() возвращает объект управления плеером:

JavaScript
const player = Vilak('#player', { kpId: 77044, type: 'serial' }); // Переключить сезон player.setSeason(2); // Переключить серию player.setEpisode(2, 5); // Удалить плеер player.destroy();

Интеграция с KinoplayerTop

Для добавления Vilak.tv как источника в KinoplayerTop, добавьте следующий код в конфигурацию плеера:

JavaScript — добавление источника в KinoplayerTop
// В конфигурации KinoplayerTop добавьте Vilak.tv как источник const kinoboxConfig = { // ... ваши настройки ... sources: [ // ... другие источники ... { name: 'Vilak.tv', url: (kpId) => `https://vilak.tv/${kpId}`, embed: (kpId) => `https://vilak.tv/${kpId}`, } ] }; // Или через iframe напрямую: const kpId = 301; // Получить из текущей страницы document.getElementById('vilak-frame').src = `https://vilak.tv/${kpId}`;

FAQ

Где найти ID Кинопоиска?

Откройте страницу фильма на kinopoisk.ru. В URL будет: kinopoisk.ru/film/301/ — число 301 и есть ID.

Что если фильм не найден?

Если контент не найден ни на одной платформе, робот автоматически ищет официальный трейлер на YouTube. Эта функция настраивается через параметр data-trailers.

Работает ли на мобильных?

Да, интерфейс полностью адаптивен. Плеер корректно работает на iOS Safari и Android Chrome.

Нужна ли регистрация или API-ключ?

Нет. Vilak.tv полностью бесплатен и не требует регистрации или API-ключей.
