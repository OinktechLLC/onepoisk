Политика конфиденциальности

Дата последнего обновления: 1 января 2025 г.

1. Сбор данных

Vilak.tv собирает минимум необходимых данных:

  • ID Кинопоиска, переданный в URL — для поиска контента
  • Технические данные запроса (IP, User-Agent) — для работы сервиса и защиты от злоупотреблений
  • Анонимная статистика использования (количество запросов, популярные ID) — для улучшения сервиса

2. Что мы НЕ собираем

  • Персональные данные (имя, email, телефон) — регистрация не требуется
  • История просмотров в привязке к личности
  • Платёжные данные

3. Cookie

Мы используем только технические cookie для сохранения настроек плеера (выбранный источник, громкость, настройки автопереключения). Маркетинговые cookie не используются.

4. Сторонние платформы

При воспроизведении видео встраивается плеер сторонней платформы (YouTube, Rutube, VK Video, OK Video). Эти платформы могут собирать свои данные согласно их политикам конфиденциальности. Vilak.tv не контролирует и не несёт ответственности за сбор данных сторонними платформами.

5. Передача данных третьим лицам

Мы не продаём и не передаём ваши данные третьим лицам, за исключением случаев, предусмотренных законодательством РФ.

6. Хранение данных

Технические логи хранятся не более 90 дней и используются исключительно для диагностики технических проблем.

7. Ваши права

Вы имеете право запросить удаление технических данных, связанных с вашим IP-адресом. Для этого напишите на privacy@vilak.tv.

8. Контакты

По вопросам конфиденциальности: privacy@vilak.tv
