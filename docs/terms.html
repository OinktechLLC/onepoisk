Дата последнего обновления: 1 января 2025 г.
Vilak.tv — легальный видеобалансер. Мы не храним, не распространяем и не пиратим видеоконтент. Весь контент берётся с официальных легальных платформ: YouTube, Rutube, VK Video, OK Video.

1. Общие положения

Используя сервис Vilak.tv («Сервис», «мы», «нас»), вы соглашаетесь с настоящими Условиями использования. Если вы не согласны с данными условиями, пожалуйста, прекратите использование Сервиса.

2. Описание сервиса

Vilak.tv предоставляет технологию поиска и встраивания легально размещённого видеоконтента с публичных платформ. Сервис:

  • Осуществляет поиск видеоматериалов по ID Кинопоиска на легальных платформах
  • Встраивает видео через официальные iframe/embed API платформ
  • Не загружает, не копирует и не перераспределяет видеоконтент
  • Не обходит систему защиты авторских прав

3. Легальность контента

Vilak.tv использует исключительно легальные источники:

  • YouTube (google.com) — официальные каналы и публично размещённый контент
  • Rutube (rutube.ru) — официальная российская видеоплатформа
  • VK Video (vk.com) — официальные каналы ВКонтакте
  • OK Video (ok.ru) — официальные каналы Одноклассников

Вы соглашаетесь использовать Сервис только в законных целях. Запрещается:

Вы можете свободно встраивать Vilak.tv на свои сайты при условии:
